# HPC Task Pareto Frontiers

This repository contains the measurement infrastructure, measurements, and figures used in the analysis of tasks from task-graph based HPC applications. All measurements target the Intel MIC architecture and rely upon the power measurement facilities provided by MPSS. 

## Directory Structure

*src* Contains the source code and build information for the runtime and example task graphs
*fig* Contains figures used in the E2SC 2016 publication
*experiments* Contains resulting data files from experiments used in E2SC 2016 publication
*scripts* Contains a set of scripts used to set up the execution environment as well as to manipulate the log files from running the scheduler
